# Skills

Skill 是**原子操作**，不代表阶段，被 Agent 或命令内部调用。

## Skill 列表

| Skill | 用途 |
|-------|------|
| module-index | 生成模块索引 |
| index-manager | 管理经验索引 |
| meta-maintainer | 维护文档元数据 |
| remind-pitfalls | 提取坑点提示 |

## Skill vs Agent

| 维度 | Skill | Agent |
|------|-------|-------|
| 定位 | 原子操作 | 阶段性角色 |
| 命名 | 动词-名词 | 拟人化名词 |
| 身份 | 无 | 有身份认同 |
| 产出 | 辅助数据 | 完整中间成果 |
