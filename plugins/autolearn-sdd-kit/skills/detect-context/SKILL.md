# detect-context

快速检索项目上下文，让 AI 了解项目状态。

## 触发场景

当用户说以下内容时自动触发：
- "帮我了解项目上下文"
- "检索一下相关经验"
- "看看有什么风险提示"
- "调用上下文侦探"

## 执行流程

1. **识别项目**：`project = basename(pwd)`

2. **匹配规则** → `./.claude/rules/risk-rules.md`
   - 扫描 trigger 关键词
   - 收集匹配的风险提示

3. **匹配经验** → `./.claude/experience/INDEX.md`
   - 按 Tag 匹配
   - 按模块匹配

4. **匹配模块索引** → `./.claude/modules/`
   - 加载相关模块的结构信息

```
需求: <需求描述>

⚠️ 风险提示:
  • <风险提示1>

📄 相关经验 (N):
  • <经验文档1>

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
